# How expressive should routing get before it's a programming language?

Slippery slope: conditions → computed goto → guards-and-variables on edges → a
Turing-complete workflow DSL nobody can analyze. Draw the line so the agent designs
*circuits* (legible, finite, checkable), not *programs*. This is the strategic
question behind the tactical [[open-question-computed-goto-representation]]; the
governing principle is [[principle-legible-circuits-not-programs]].
